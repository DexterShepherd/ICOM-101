Flex Box

Flex box lets us position elements within a container with greater control than we had with just the box model + floats.

For flex to work, we need a container element with display: flex css rule applied.

Any child elements of the flex container will be positioned with the flex rules of the container.

Embedded content

We're entering the wild west a bit here. Different sites embed content in different ways, so you might need to mess around a bit to get something you like.

Youtube

find a video you like, and look for the share option in the bottom left corner

Click the embed option from the share menu and copy the embed code.

Vimeo

The idea with most of these things is pretty similar. You look for the share button, then find the embed code to copy.

embed codes usually start with an iframe element

SoundCloud

Embedding videos in in a way that works with different screen sizes is notoriously complicated. I'm of the opinion that a nice image that links to the video is often a better approach. If you absolutely need to embed something responsively, check out Embed Responsively. They provide an alternate embed code that sometimes looks nicer.

Psuedo-selectors

In addition to the tag/class/descendant selectors we looked at last week, css gives psuedo-selectors.

These allow us to select an element based on a few more complex rules.

There are many psuedo-selectors, most are specific to certain uses, but some of the more general ones are

  • :first-child / :last-child
  • :first-of-type / :last-of-type
  • :hover
  • :active
